 Overview
This repository contains Linux shell scripts and basic command outputs created as part of the Linux assignment. It includes demonstrations of directory backup, CPU/Memory monitoring, and automated file downloading.

Scripts Included
   1. backup.sh
Creates a timestamped backup of a directory.

  2. monitor.sh
Logs CPU and memory usage to a log file.

  3. download.sh
Downloads a file from the internet using wget.

------------------------------------------------------------------------------------------------------------------------------------------
 How to Run the Scripts
1. Make script executable

 chmod +x scriptname.sh
2. Run the script

3. ./scriptname.sh
Screenshots of script execution and outputs are available in the repository.
